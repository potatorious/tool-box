# Tool Box Lab

AdSense review risk is lower when a utility site provides a real tool, original explanatory content, clear navigation, contact information, and privacy/legal pages. This project is a static Korean utility site built around a time/rate/value calculator.

## Included

- Working calculator with browser-only computation
- Usage guide, calculation standards, and FAQ
- About, Contact, Privacy Policy, Terms, and Editorial Policy pages
- Basic SEO metadata, robots.txt, and sitemap.xml

## Before Deployment

Replace `contact@example.com` with a real domain email address.

After receiving an AdSense publisher ID, add a valid `ads.txt` file:

```txt
google.com, pub-YOUR_PUBLISHER_ID, DIRECT, f08c47fec0942fa0
```

Update canonical URLs and sitemap locations to absolute production URLs after the final domain is chosen.
